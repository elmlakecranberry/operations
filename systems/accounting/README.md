# Accounting Systems

Documentation for financial software, bookkeeping tools, and accounting workflows.

## What Belongs Here

- Software used for bookkeeping and accounting
- How to access the accounting system
- Common tasks (invoicing, expense entry, reconciliation)
- Report generation procedures
- Integration with other systems (payroll, banking)
- Vendor/support contact information

## What Does NOT Belong Here

- Financial policies or approval thresholds (management repo)
- Budget amounts or financial targets (management repo)
- Account numbers, credentials, or banking details (never commit)
- Audit findings or compliance assessments

## Documentation to Add

When documenting accounting systems, include:

- System name and vendor
- How to log in
- User roles and permissions overview
- Common workflows (step-by-step)
- How to get help or report issues
- Backup and data export procedures
