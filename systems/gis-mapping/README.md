# GIS & Mapping Systems

Documentation for geographic information systems, mapping tools, and spatial data management.

## What Belongs Here

- GIS software and platforms used
- How to access mapping systems
- Creating and editing maps
- GPS/GNSS equipment operation
- Data collection procedures
- Map printing and export
- Integration with other systems
- Vendor/support contact information

## What Does NOT Belong Here

- Property boundary disputes or legal matters
- Confidential survey data
- Decisions about land use (management repo)

## Documentation to Add

When documenting GIS/mapping systems, include:

- Software names and vendors
- How to log in and access data
- Coordinate system and projection standards
- How to add or update features
- Data backup procedures
- Field data collection workflows
- How to generate reports or printable maps
- How to get help or report issues
