# File Storage Systems

Documentation for document management, file sharing, and backup systems.

## What Belongs Here

- File storage services and how to access them
- Folder structure and organization
- Sharing and permissions (how to set them, not policy)
- Backup procedures
- How to find archived documents
- Mobile access instructions
- Vendor/support contact information

## What Does NOT Belong Here

- Data retention policies (management repo)
- What documents require approval to share
- Confidential document classifications
- Compliance requirements

## Documentation to Add

When documenting file storage systems, include:

- Service name and vendor
- How to access (desktop, web, mobile)
- Folder naming conventions
- How to share files internally
- How to share files externally
- How to request more storage
- Backup frequency and recovery procedures
- How to get help or report issues
