# Payroll & HR Systems

Documentation for payroll processing, timekeeping, and human resources tools.

## What Belongs Here

- Payroll software and how to use it
- Timekeeping system operation
- How to process a pay run
- How to onboard a new employee (system steps)
- How to offboard an employee (system steps)
- Benefits enrollment procedures
- Vendor/support contact information

## What Does NOT Belong Here

- Pay rates, salary information, or compensation policies (management repo)
- Personnel decisions or performance management policies
- Employee personal information
- Confidential HR matters

## Documentation to Add

When documenting payroll/HR systems, include:

- System name and vendor
- How to log in
- Pay period schedule
- Deadline for timecard submission
- Steps for running payroll
- How to handle corrections or adjustments
- Year-end procedures (W-2s, tax filings)
- How to get help or report issues
