# Communications Systems

Documentation for email, messaging, phone systems, and collaboration tools.

## What Belongs Here

- Email setup and configuration
- Messaging/chat platforms
- Phone system operation
- Video conferencing tools
- Contact lists and directories
- Mobile device setup
- Vendor/support contact information

## What Does NOT Belong Here

- Communication policies (what to say, when to escalate)
- Confidential contact information
- Emergency notification policies (procedures go in runbooks)

## Documentation to Add

When documenting communication systems, include:

- Service name and vendor
- How to access and log in
- Setting up email on new devices
- Using shared mailboxes or aliases
- Voicemail setup and retrieval
- Conference call and video meeting procedures
- How to update contact information
- How to get help or report issues
