## Column Definitions

- **Parcel ID** — Internal name (should match parcels.md)
- **Tax Key / PIN** — Official parcel identification number
- **Land Value** — Assessed value of land only
- **Improvement Value** — Assessed value of buildings/improvements
- **Total Assessment** — Combined assessed value
- **Classification** — Agricultural, residential, commercial, etc.
- **Exempt?** — Yes/No — any tax exemptions in place
- **Notes** — Assessment year, pending appeals, special circumstances
