# Buildings

Inventory of structures at Elm Lake Cranberry.

## Guidance

Include any structure that:

- Has a roof and walls
- Requires maintenance or insurance
- Houses equipment, people, or operations

For each building, document what it's used for, how it's heated/cooled, and any critical systems it contains.

## Column Definitions

- **Name** — Common name used to refer to this building
- **Type** — Shop, barn, residence, storage, office, etc.
- **Primary Use** — What happens here day-to-day
- **Heating System** — Propane, electric, wood, none
- **Cooling System** — AC, fans, none
- **Electricity** — Yes/No, and service size if known
- **Water** — Well, municipal, none
- **Critical Systems** — Anything inside that would cause operational problems if it failed (freezers, compressors, controls, etc.)
- **Notes** — Anything else relevant
